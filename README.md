# Number Guessing Game

> A simple Python CLI-based game demonstrating control flow, user interaction, and random number generation.

## Features
- Random number generation between 1 and 100
- User input handling
- Feedback for each guess (Too high / Too low)
- Tracks number of attempts

## Tech Stack
- Python
- random module

---

## How It Works
The program generates a random number, and the user keeps guessing until the correct number is found. After each guess, feedback is provided to guide the user.
